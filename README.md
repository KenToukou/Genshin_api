## プロジェクトについて

このコードは原神のキャラクター情報をデータベースに登録するAPIである
実行手順:
  compose-up-buildでコンテナを起動
  localhost:8000/docsでFASTAPIのSUAGGERへ移動
  APIでサーバーとやり取りが可能になる。
  *DBへ情報を登録する際は、以下の手順で情報を登録する必要がある
    1 : Countryテーブルに国の情報を登録 (例: モンド)
    2 : 所属組織テーブルに情報を登録 (例: 西風騎士団)
    3 : 属性テーブルに情報を登録 (例: 風元素)
    4 : 名前テーブルに情報を登録 (例: ジン)
